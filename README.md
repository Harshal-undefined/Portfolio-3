## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `portfolio`)
2. Push all files to the `main` branch
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch, root `/`
5. Save — your site will be live at `https://[username].github.io/portfolio/`

## ✏️ Customizing

- Edit contact info in `index.html` (email, phone)
- Swap images in `/images/` folders
- Update project descriptions in `/projects/*.html`
- Colors are CSS variables in `css/style.css` under `:root`
